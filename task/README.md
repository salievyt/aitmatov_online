# Задачи по оптимизации Aitmatov Digital

## Контекст
Продуктовый аудит выявил критические проблемы (оценка 52/100):
- Отсутствие аналитики и метрик
- Отсутствие монетизации
- Низкое удержание пользователей
- Отсутствие родительского контроля

## Структура задач
Задачи распределены по ролям:
- `senior-product-manager/` - продуктовая стратегия и метрики
- `flutter-senior-architect/` - архитектурные решения
- `code-reviewer/` - ревью и качество кода
- `senior-qa-engineer/` - тестирование и QA

## Приоритеты
- **P0 (Critical)** - блокирует запуск, решать немедленно
- **P1 (High)** - критично для успеха, 0-3 месяца
- **P2 (Medium)** - важно, 3-6 месяцев
- **P3 (Low)** - желательно, 6-12 месяцев

## Целевые метрики
- D1 Retention: 50-60%
- D7 Retention: 30-40%
- Конверсия в платных: 3-5%
- ARPU: $0.50-1/месяц
- CAC/LTV: 1:3
