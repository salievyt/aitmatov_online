# P1: Родительский контроль и мониторинг

**Приоритет:** P1 (High)  
**Срок:** 4-8 недель  
**Роль:** Senior Product Manager  
**Зависимости:** flutter-senior-architect/parent-control-architecture.md

## Проблема
Отсутствие родительского контроля — барьер доверия для родителей:
- Родители не могут отслеживать прогресс детей
- Нет контроля времени использования приложения
- Нет отчётов об успеваемости
- Родители не вовлечены в образовательный процесс

## Цели
1. Создать родительский кабинет для мониторинга прогресса детей
2. Увеличить конверсию в Premium через родителей (B2C канал)
3. Повысить доверие к продукту
4. Снизить churn через вовлечение родителей

## Функции родительского контроля

### 1. Родительский кабинет (Dashboard)

#### Обзор прогресса
- **Активность**: дни активности за неделю/месяц
- **Стрик**: текущий стрик ребёнка
- **Курсы**: начатые и завершённые курсы
- **Время обучения**: часы в приложении (по дням)
- **Достижения**: разблокированные бейджи и уровень

#### Детальная статистика
- **По предметам**: прогресс по математике, физике, литературе и т.д.
- **Результаты квизов**: средний балл, динамика
- **Слабые места**: темы, требующие внимания
- **Сильные стороны**: темы с высокими результатами

#### Сравнение
- Прогресс ребёнка vs средний по классу
- Прогресс vs прошлый месяц
- Топ-3 предмета ребёнка

### 2. Уведомления для родителей

#### Email отчёты
- **Еженедельный отчёт** (воскресенье, 19:00):
  - Активность за неделю
  - Завершённые уроки и курсы
  - Достижения
  - Рекомендации

- **Месячный отчёт** (1-го числа):
  - Прогресс за месяц
  - Сравнение с предыдущим месяцем
  - Сертификаты и достижения

#### Push/SMS уведомления (опционально)
- **Milestone достижения**: "Ваш ребёнок завершил курс 'Математика 7 класс'!"
- **Стрик под угрозой**: "Стрик вашего ребёнка (14 дней) под угрозой"
- **Низкая активность**: "Ваш ребёнок не заходил в приложение 3 дня"
- **Отличные результаты**: "Ваш ребёнок получил 100% в квизе по физике!"

### 3. Контроль времени (Screen Time)

#### Лимиты времени
- Установка дневного лимита (например, 2 часа/день)
- Установка временных окон (например, 18:00-21:00)
- Блокировка приложения вне разрешённых часов

#### Мониторинг
- График времени использования по дням
- Среднее время сессии
- Самое активное время дня

### 4. Контроль контента

#### Разрешённые курсы
- Родитель может выбрать, какие курсы доступны ребёнку
- Блокировка определённых категорий (если есть)

#### Мессенджер
- Просмотр списка классов/групп ребёнка
- Опция отключения мессенджера (только обучение)

### 5. Цели и мотивация

#### Установка целей
- Родитель устанавливает цели: "Завершить 3 курса в месяц"
- Ребёнок видит цели в своём профиле
- Прогресс к цели в реальном времени

#### Система наград (опционально)
- Родитель может установить награды за достижения
- Например: "Завершишь курс — пойдём в кино"
- Чек-лист наград в приложении

### 6. Связь родитель-ребёнок

#### Сообщения поддержки
- Родитель может отправить сообщение поддержки в приложение
- Ребёнок видит сообщение после завершения урока
- Примеры: "Горжусь тобой!", "Продолжай в том же духе!"

#### Совместные достижения
- Родитель получает уведомление о достижениях ребёнка
- Возможность поделиться достижением в соцсетях

## User Flow

### Регистрация родительского аккаунта

**Вариант 1: Родитель регистрируется отдельно**
1. Родитель скачивает приложение
2. Выбирает "Я родитель"
3. Регистрируется (email/phone)
4. Добавляет ребёнка по коду/email
5. Ребёнок подтверждает связь

**Вариант 2: Ребёнок приглашает родителя**
1. Ребёнок в настройках выбирает "Пригласить родителя"
2. Отправляет приглашение (email/SMS)
3. Родитель переходит по ссылке
4. Регистрируется и автоматически связывается с ребёнком

### Ежедневное использование
1. Родитель открывает приложение
2. Видит дашборд с прогрессом ребёнка
3. Проверяет активность за день
4. Читает уведомления о достижениях
5. (Опционально) Отправляет сообщение поддержки

## Монетизация через родителей

### Premium для родителей
Родительский контроль — **Premium feature**:
- Free tier: базовый обзор прогресса (только текущая неделя)
- Premium: полная аналитика, отчёты, контроль времени, цели

### Value proposition для родителей
- "Следите за успехами вашего ребёнка"
- "Помогите ребёнку учиться эффективнее"
- "Контролируйте время использования"
- "Получайте еженедельные отчёты"

### Конверсия
- Родители более склонны платить за образование детей
- Целевая конверсия родителей: **10-15%** (выше, чем у детей)
- ARPU родителей: $1-2/месяц (выше, чем у детей)

## Метрики успеха

### Adoption
- % детских аккаунтов, связанных с родительскими: 30-40%
- % родителей, активирующих Premium: 10-15%

### Engagement
- % родителей, заходящих в приложение еженедельно: 60%
- Open rate email отчётов: 40-50%

### Retention
- Churn rate детей с подключёнными родителями: -30% (ниже, чем без родителей)
- Retention детей с активными родителями: +20%

### Monetization
- Конверсия родителей в Premium: 10-15%
- ARPU родителей: $1-2/месяц

## Риски и митигация

### Риск 1: Дети сопротивляются контролю
**Митигация**:
- Прозрачность: ребёнок видит, что видит родитель
- Позитивный фрейминг: "Родители гордятся твоими успехами"
- Опция отключения (с согласия родителя)

### Риск 2: Родители слишком контролируют
**Митигация**:
- Рекомендации по здоровому использованию
- Фокус на поддержке, а не наказании
- Образовательный контент для родителей

### Риск 3: Privacy concerns
**Митигация**:
- Чёткая политика конфиденциальности
- Ребёнок должен подтвердить связь с родителем
- Родитель видит только образовательные данные (не сообщения)

### Риск 4: Низкий adoption
**Митигация**:
- Промо родительского контроля в onboarding
- Email кампании для родителей
- Партнёрство со школами (родительские собрания)
