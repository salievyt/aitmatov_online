Это мобильное приложение «Айтматов онлайн» — 
образовательная платформа для школьников и учителей,
построенная на Flutter с использованием архитектуры 
BLoC для управления состоянием, Go Router для навигации
и Dio для работы с сервером. Приложение позволяет ученикам 
проходить курсы по разным предметам (видео, аудио и текстовые уроки),
смотреть расписание занятий, общаться в групповых чатах, отслеживать 
свой прогресс и изучать тематический раздел «Мир Айтматова». 
Учителям доступен личный кабинет с аналитикой по классам и инструментами
для управления контентом, а администраторам — панель управления пользователями,
расписанием и общей статистикой платформы. Вся бизнес-логика построена на
чистой архитектуре с разделением на слои данных, домена и презентации, а 
зависимости управляются через GetIt с кодогенерацией Injectable.
